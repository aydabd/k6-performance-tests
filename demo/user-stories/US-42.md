# US-42: Browse Dog Breeds

**As a** dog enthusiast using the Dog CEO API,
**I want to** retrieve a list of all dog breeds,
**So that** I can display them to users and let them explore random images.

## Acceptance Criteria

1. `GET /api/v2/breeds/list/all` responds with HTTP 200.
2. Response body contains a `message` object with breed names as keys.
3. Response time p95 is under 500 ms.
4. Error rate is below 1%.

## Notes

- The API is public and requires no authentication.
- Test with at least 10 virtual users for 30 seconds.
